获取你的专属 Token

访问 keenotes.afoo.me 点击 FREE Trial。 在 Waiting List 页面输入邮箱，你将收到一封包含 Token 的授权邮件。

注意： Token 是你设备间通信的唯一令牌，请妥善保管。
2

下载全平台客户端

前往 GitHub Releases 页面下载适合你设备的版本。

  • 支持 macOS (Intel/M1) & Windows & Linux
  • 支持 iOS & Android
为何没有 Web 端？
为了确保绝对的端到端加密 (E2EE)，Web 端存在潜在安全风险，因此我们坚持使用原生客户端。
3

配置与同步

启动程序，在 Settings 界面输入你的 Token。

关键一步： 设置一个只有你自己知道的密码（二次确认）。点击 Save 后，你的安全笔记之旅正式开始。

只要 Token 一致，多端自动同步
