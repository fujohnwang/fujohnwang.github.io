1 准备与运行

  1. 先装virtualbox, boot2docker需要它！

  2. download and install

    • 更简单的做法：brew update & brew install boot2docker

  3. boot2docker init to download vm iso and create it in virtual box;

    ➜  ~  boot2docker init
Latest release for boot2docker/boot2docker is v1.6.0
Downloading boot2docker ISO image...

  4. boot2docker start

    • 返回信息中会提示你设置环境变量，不用手动设置， 直接执行第4步的命令即可。

  5. boot2docker shellinit - 设置相关环境变量

2 常用命令说明

2.1 升级boot2docker

下载最新的boot2docker安装包并安装即可。

2.2 升级docker

运行boot2docker download

2.3 下载最新的boot2docker iso

boot2docker download

2.4 删除虚拟机景象

boot2docker delete
